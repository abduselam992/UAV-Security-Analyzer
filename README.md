# UAV Security & Flight Log Analyzer

## Overview
This project was developed as part of my application for the **INSA Drone Learning Program**. It is a forensic tool designed to analyze drone telemetry data for safety and security anomalies.

## Key Features
* **GPS Spoofing Detection:** Identifies sudden coordinate jumps that suggest signal interference.
* **Regulatory Compliance:** Automatically flags altitude violations (above 122m/400ft).
* **Battery Health Audit:** Ensures missions conclude with safe power margins.

## How to Run
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/UAV-Security-Analyzer.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the script: `python analyzer.py`

## Goal
To contribute to Ethiopia's national security by ensuring Unmanned Aerial Systems (UAS) operate within legal and secure parameters.
